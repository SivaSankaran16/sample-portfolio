PROJECTS

In this exciting project, we will create a Nerf turret equipped with computer vision capabilities using OpenCV. The turret will be able to autonomously track and target objects using a webcam or camera module, making it an interactive and engaging device for both fun and educational purposes. This project combines elements of robotics, computer vision, and mechanical design to bring a playful twist to target shooting.

The Online Gas Booking System is a web-based application that aims to streamline the process of booking gas cylinders for residential and commercial customers. The system provides a convenient platform for users to place gas cylinder orders online, check their booking status, and manage their account details. It eliminates the need for customers to visit the gas agency in person and simplifies the entire booking process..

The Python Tic-Tac-Toe game is a classic two-player board game where players take turns marking either "X" or "O" on a 3x3 grid. The objective is to be the first player to get three of their marks in a horizontal, vertical, or diagonal row..

The Basic Hotel Booking System in Python is a simple console-based application that allows users to browse available hotel rooms, make reservations, and view their booking details. This project provides an introduction to handling data structures, user input, and basic logic using Python.
